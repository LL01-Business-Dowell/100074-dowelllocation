How to Query

Have fun!!!

Definitions

Type of querying factor

Value according to query-type eg India, PAK, KEN, 234

Examples

  • name : Query by country name or city name (India, Kenya, Pakistan, Nigeria etc) or (Mumbai, Nairobi, Cape Town etc)
  • code : Query by country code or city code (91, 254, 92, 234 etc) or (MUM, NBO, CPT etc)
  • short : Query by country short (IND, KEN, PAK, NGA etc)

username

Username of user using the software or type of programmer using the api.

session-id

Session Id from Login Fucntion

project-code

Project code eg 100074, 100018, 100025
